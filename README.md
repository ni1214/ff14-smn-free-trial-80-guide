# FF14 Free Trial Lv80 Guide

スマホ向けに見やすく整理した、FF14フリートライアルLv80の二人用攻略アプリです。

公開ページ:

- https://ni1214.github.io/ff14-smn-free-trial-80-guide/

内容:

- タブ切り替え式のスマホUI
- マイキャラ欄でジョブ/レベル/メイン進行/詩学/Lodestoneリンクを管理
- 全ジョブのレベル別スキル回し
- 全ジョブの全スキル一覧、習得Lv、ジョブクエスト要否、公式アイコン表示
- 単体 / 3体以上AoE / 開幕バーストの切り替え
- 公式スキルアイコン付き
- Lv/ジョブ別の装備チェッカー
- 召喚士Lv50のガーロンドRE装備名/アイコン完全一致表示
- Lv20など低レベル帯の具体的なビギナー装備導線
- Lv50以降の詩学装備と詩学集め導線
- メイン進行に合わせた公式画像切り替え
- Lv/ジョブ/メイン進行から次の解放要素を出すナビ
- 今日やること、買い物リスト、迷子防止メモ、ごほうび候補を楽しみナビに統合
- ジョブ/レベル/進行、詩学、チェック状態のスマホ内保存
- ID/討滅戦の初見ギミック即答ナビ
- 新生〜漆黒Lv80までのID/討滅戦リスト
- マイキャラ欄からLodestoneプロフィールをワンタップで開く導線
- 外部リンクは日本語ページのみ
- 出典リンク付き

## 自宅PCで続きを開発する

最初に読むファイル:

- `AGENTS.md`: AIやCodexに守らせたい開発ルール
- `DEVELOPMENT.md`: GitHub Desktopでcloneして公開する手順

PowerShellで公開する場合:

```powershell
.\tools\publish.ps1 -Message "変更内容を短く書く"
```

GitHub Pagesは `main` ブランチへpushすると自動で公開されます。

## 参考にした設計

大手FF14サイトの強みを、二人用の小さいサイトでも使える形に落とし込んでいます。

- 大手攻略サイト: ジョブ別に専門ガイドを置き、目的別に探しやすくする
- 公式プレイガイド: 解放条件や遊び方を日本語ページへ絞る
- エオルゼアデータベース: クエスト、アイテム、装備名の確認に使う
- 公式ジョブガイド: スキル名と公式アイコン取得に使う
