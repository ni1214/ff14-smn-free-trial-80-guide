# FF14 Summoner Free Trial Lv80 Guide

スマホ向けに見やすく整理した、FF14フリートライアルLv80の二人用攻略アプリです。

公開ページ:

- GitHub Pagesで配信

内容:

- タブ切り替え式のスマホUI
- ジョブ/レベル/メイン進行を1か所で指定する共通設定
- 全ジョブのレベル別スキル回し
- 単体 / 3体以上AoE / 開幕バーストの切り替え
- 公式スキルアイコン付き
- Lv/ジョブ別の装備チェッカー
- 召喚士Lv50のガーロンドRE装備名/アイコン完全一致表示
- Lv20など低レベル帯の具体的なビギナー装備導線
- Lv50以降の詩学装備と詩学集め導線
- メイン進行に合わせた公式画像切り替え
- Lv/ジョブ/メイン進行から次の解放要素を出すナビ
- 今日やること、買い物リスト、迷子防止ナビ、ごほうび図鑑
- ジョブ/レベル/進行、詩学、チェック状態のスマホ内保存
- ID/討滅戦の初見ギミック即答ナビ
- 新生〜漆黒Lv80までのID/討滅戦リスト
- Lodestoneプロフィールをすぐ開けるキャラクター確認メモ
- 外部リンクは日本語ページのみ
- 出典リンク付き

## 参考にした設計

大手FF14サイトの強みを、二人用の小さいサイトでも使える形に落とし込んでいます。

- 大手攻略サイト: ジョブ別に専門ガイドを置き、目的別に探しやすくする
- 公式プレイガイド: 解放条件や遊び方を日本語ページへ絞る
- エオルゼアデータベース: クエスト、アイテム、装備名の確認に使う
- 公式ジョブガイド: スキル名と公式アイコン取得に使う
