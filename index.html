まず結論

召喚士の覚え方はこの4段階
Lv1-57 基礎魔法 + 3属性 Lv58-69 トランス・バハムート Lv70-79 バハムート Lv80 フェニックス追加
ざっくり言うと、
Lv10-50は「エーテルチャージ → 召喚3種 → 基本魔法」、
Lv60は「トランス・バハムート → 召喚3種」、
Lv70は「シアリングライト → サモン・バハムート → 召喚3種」、
Lv80は「バハムート → 召喚3種 → フェニックス → 召喚3種」です。
最初に覚える3つ
  • 単体はだいたい「通常魔法」より「属性中の専用魔法」が強い。
  • 3体以上ならAoEへ切り替える。単体魔法を混ぜすぎない。
  • 召喚士は「次に押せる強いボタンを腐らせない」だけでかなり火力が出る。

成長マップ

レベルシンクで急に押すボタンが減っても、この節目だけ覚えておけば戻しやすいです。

共通ルール

単体 / 少数戦 1〜2体は単体回し。まずは強化フェーズや召喚を優先して、空いたGCDを通常魔法で埋めます。
AoEへ切り替える基準 3体以上ならAoEへ。The Balanceのレベリングガイドも「3体以上」をAoE基準にしています。
召喚の順番 Lv80までは順番の縛りはそこまで強くありません。長詠唱のイフリートは足を止められるとき、動きたいときはタイタンやガルーダ寄りにすると楽です。
この資料の前提 高難度向けの最適化よりも、フリートライアル / レベリング / レベルシンクで迷わず押せることを優先しています。

ユーティリティの使いどころ

出典

回しの優先度は主にThe Balanceの召喚士レベリングガイド、アクション名・修得レベル・アイコンは公式ジョブガイド、フリートライアルLv80拡張はPatch 7.5パッチノートを参照しています。
なお「Lv80でバハムートとフェニックスを交互に回す」というまとめは、公式の共有リキャスト仕様とレベリングガイドの優先度記述をあわせて整理した要約です。
